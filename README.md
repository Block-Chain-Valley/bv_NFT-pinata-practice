### ```.env``` 파일 생성후 환경변수 입력
root 폴더에 ```.env``` 파일을 생성하고 
```shell
SEPOLIA_PRIVATE_KEY="본인의 메타마스크 privatekey"
POLYGON_API_URL = "Alchemy에서_받은_polygon_api_url"
```
을 작성해주세요.
